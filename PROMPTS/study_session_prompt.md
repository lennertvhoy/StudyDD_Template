:books: Study Session Prompt

Run a focused StudyDD study session.

## Setup

1. Read the current study state.
2. Identify the active focus and the weakest skill or pending topic.
3. Select 3–5 questions aligned with that focus.

## During the session

1. Ask one question at a time.
2. State the active question ID and expected answer format.
3. Wait for the learner's answer.
4. Grade the actual answer against the answer key.
5. Explain the verdict in plain language.
6. If needed, ask a repair question.
7. Close the question before opening the next.

## Question selection rules

- Prioritize weak areas.
- Mix conceptual and applied questions.
- Match the difficulty to the learner's current readiness estimate.
- Avoid asking the same question repeatedly.

## End of session

1. Summarize what was covered.
2. List evidence items.
3. Identify weak areas.
4. Propose updates to the skill map and state.
5. Propose the next study action.
