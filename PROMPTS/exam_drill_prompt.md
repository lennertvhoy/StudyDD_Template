:target: Exam Drill Prompt

Run a realistic exam-style drill inside StudyDD.

## Setup

1. Read the current study state and skill map.
2. Select a balanced set of questions covering weak areas, practiced areas, and confirmed strengths.
3. Simulate exam conditions: clear wording, no hints unless asked.

## During the drill

1. Ask one question at a time.
2. Do not reveal the answer until the learner has responded.
3. Grade strictly against the answer key.
4. Record every result as evidence.

## Grading

- **correct** — fully meets the answer key
- **partial** — partly correct but missing something important
- **incorrect** — does not meet the answer key

Be strict. Exam drills exist to find gaps, not to make the learner feel good.

## End of drill

1. Report the score.
2. Highlight weak areas that need more work.
3. Confirm strengths that held up under exam pressure.
4. Propose updates to readiness estimates.
5. Propose the next action.
