# SOURCE_INDEX — Trusted Source Tracking

> **Agent-maintained.** Record sources before building or revising a skill map.

## Source trust policy

1. Prefer official exam guides, vendor documentation, course syllabi, instructor materials, or employer-owned source systems.
2. Treat old notes, spreadsheets, exports, blog posts, and generated summaries as secondary until verified.
3. When sources conflict, record the conflict and choose the most authoritative current source.
4. Do not build readiness claims from source coverage alone. Readiness requires learner evidence.
