# SESSION_LOG — Study Session History

> **Agent-maintained.** The coding agent appends one entry per session. The learner may inspect or override it.

## Format

Each entry should include:

- **Date:** ISO 8601 date
- **Focus:** topic or skill practiced
- **Questions asked:** list of question IDs
- **Result summary:** what was demonstrated
- **Evidence added:** references to `EVIDENCE_LOG.md`
- **State changes:** what changed in `STUDY_STATE.yaml` or `SKILL_MAP.yaml`
- **Next action proposed:** what the learner should do next
